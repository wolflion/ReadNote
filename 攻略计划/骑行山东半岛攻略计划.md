##  攻略
###  装备准备
+ 山地车：就用我研一买的*乐途26寸24速*自行车吧，一直在信院门口吃灰呢。
+ 骑行头盔：*TOSUOD*，还顺带购买这个牌子的多功能工具
+ **我玩户外的装备骑行可以用的**
	+ 头巾，头灯(手电)，快干衣，水袋，
+ **需要后续买的**
	+ 链条油
	+ 半指手套
	+ 骑行裤
	+ 补胎装备，以及备胎

###  具体行程
+ 青岛到即墨；烟台海阳市凤城镇，102公里。再骑行40公里到乳山市区。
	+ 主要沿着S293骑，**丁字湾跨海大桥**，就是S239结束了。
	+ 凤城到乳山，主要骑S202。
