##  Part2 个人领域的成功：从依赖到独立 52
###  chap3 习惯一 积极主动-个人愿景的原则
+ *弗兰克尔*人生有三种主要的价值观，一是经验价值观，来自自身经历；二是创造价值观，源于个人独创；三是态度价值观，即面临绝症等困境时的回应。
+ **积极主动**不仅指行事的态度，还意味着为人一定要对自己的人生负责。能够慎重思考，选定价值观并将其作为自己行为的内在动力；即使生活受到了外界条件的制约，那也是我们有意或无意地选择了被外界条件控制，这种选择称为 *消极被动*。
+ 积极主动的人专注于 **“影响圈”**，他们专心做自己力所能及的事；消极被动的人则全神贯注于“关注圈”，紧盯他人弱点，环境问题以及超出个人能力范围的事情不放。
+ **“由内而外”地改变**先改变个人行为，让自己变得更充实，更具创造力，然后再去施加影响，改变环境。
+ *对错误的回应*
+ *两种能够直接掌控人生的途径：*一是做出承诺，并信守承诺；二是确立目标，并付诸实践。
+ **实践P67**
	+ 观察自己说话的方式，确认自己消极还是积极。
	+ 找到令自己备感挫折的问题，判断属于直接控制，间接控制还是无法控制的问题，找出解决问题的第一个步骤。

###  chap4 习惯二 以始为终-自我领导的原则
+ **领导与管理：**管理关注基层，思考的是“怎样才能有效地把事情做好”；领导关注高层，思考的是“我想成就的是什么事业”。
+ *P81*的图不错。
+ **以原则为中心**的人会保持冷静和客观，不受情绪或其他因素的干扰，纵观全局，深思熟虑后才做出正确的选择。
+ **唯有参与，才有认同**
+ *实践P97*
	+ 检讨自己对目前所扮演的角色和自己的生活是否满意。
	+ 个人使命宣言

###  chap5 习惯三 要事第一-自我管理的原则
+ *人类特有的四大天赋：*想象力，良知，独立意志以及最为重要的自我意识。
+ **独立意志**指的是做出决定和主动选择，并根据这些决定和选择采取具体行动的能力。
+ *有效的自我管理*所涉及的大都是左脑所擅长的能力：分解，分析，排序，具体运用以及在规定时间内完成任务等。
+ *自我管理的实质*就是自律和条理，是对计划的实施。
+ 按照彼得.德鲁克的观点，高效能人士的脑子里装的不是问题，而是机会。 *他们的思维是预防型的，总是能够做到防患于未然。*
+ 以第二类事务为中心的日程安排需要以下四项关键步骤(P109开始的例子)：
	+ 确认角色
	+ 选择目标
		+ 思考下一个周计划中每一任务栏下你最想做的一两件事，作为你选定的目标。
	+ 安排进度
	+ 每日调整
+ 对人不可讲效率，对事才可如此。对人应讲效用--即某一行为是否有效。
+ 授权分为两类
	+ 指令型授权：让别人“去做这个，做完告诉我”。
	+ 责任型授权：关注重点是最终的结果。在以下五方面达成清晰，坦诚的共识
		+ 预期成果
		+ 指导方针
		+ 可用资源
		+ 责任归属
		+ 明确奖惩
+ *实践P117*
	+ 每周都制订一个周计划，并规定一个具体的时间专门做这件事情。
