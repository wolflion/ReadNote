+ 作者是 **圣严法师**
+ ISBN：978-7-5675-0287-1

##  内容
#### 1. 四圣谛讲记
+ 四圣帝的三个层次
	+ 此是“苦”的事实；苦果定有其苦因，名为“集”；此是灭苦之“道”；此是苦“灭”的涅槃。
	+ 苦宜灭，集宜断，道宜修，灭宜证。
	+ 苦已灭，集已断，道已修，灭已证。
+ 基础的佛法有以下四项
	+ 依戒而住
	+ 依法为师   **三法印：**诸行无常，诸法无我，涅槃寂静。
	+ 以解脱为目的
	+ 以四圣谛为总纲
+ **五蕴**：色，受，想，行，识。
+ **八苦**：生，老，病，死，爱别离，怨憎会，求不得，以及五蕴炽盛苦。
+ 苦的事实是苦谛，苦的原因是即谛。

#### 2. 六波罗蜜讲记
+ 只要是无为无漏，有为无漏的佛法，都叫作 **波罗蜜**；所谓无漏，就是无我的，无私的。
	+ 用利他的方式来利己，就是做事的发心。{而不只是用自利的方式来利己}
+ 六波罗蜜
	+ 布施
	+ 持戒
	+ 忍辱
	+ 精进
	+ 禅定
	+ 般若

#### 3. 四弘誓愿讲记
+ 众生无边誓愿度
+ 烦恼无尽誓愿断
+ 法门无量誓愿学
+ 佛道无上誓愿成


#### 4. 四念处讲记
+ 正确的修行方法，可以帮助我们在任何情况下都能感觉身心的自在。
+ 虽然修习四念处不是为了禅定，但当有了禅定，四念处真正的目标--智慧，就会随之而来。
+ 四念处
	+ 身念处
	+ 受念处
	+ 心念处
	+ 法念处
+ **真正要了解无常并不是靠理解，而是要体验的。**

#### 5. 四正勤讲记
+ **四正勤：**四种正确的勤劳和精进。
	+ 已生恶法为除断，未生恶法不令生，未生善法为生，已生善法为增长。

#### 6. 四如意足讲记
+ 四如意足内容
	+ 欲如意足
	+ 精进如意足
	+ 心如意足
	+ 思惟如意足

#### 7. 五根五力讲记
+ **五根：**
	+ 一类是：生理机能的眼，耳，鼻，舌，身，物质的五种色根。
	+ 二类是：道品次第的信，精进，念，定，慧，为五种无漏法之根基。
+ **五力：**信力，精进力，念力，定力，慧力。

#### 8. 七觉支讲记
+ **三十七道品**称为道品次第。【四念处；四正勤；四如意足；五根；五力；七觉支；八正道。】
+ **七觉支：**念觉支，择法觉支，精进觉支，喜觉支，除觉支，舍觉支。
	+ **不要把没有原则的随便，误以为是方便。**

#### 9. 八正道讲记
+ 八正道就是帮助我们解脱两大问题的方法：
	+ 1. 如何度过种种的困难而不自扰扰人
	+ 2. 面临死亡时，不会恐惧，害怕，无奈
+ **八正道：**正见；正志；正语；正业；正命；正方便；正念；正定。
